# Team Task Manager

## Overview
This is a full-stack web application developed by me to manage team projects and tasks with role-based access control (Admin / Member).

Users can create projects, assign tasks, and track progress with a clean dashboard.

---

## Features
- User Authentication (JWT + bcrypt)
- Role-based access (Admin / Member)
- Project creation and management
- Task assignment and status tracking
- Dashboard with task overview and overdue tasks

---

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: PostgreSQL (via Railway)
- Authentication: JWT + bcrypt
